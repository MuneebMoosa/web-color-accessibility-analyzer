# See Easy - Color Accessibility Assistant

## Overview
The **See Easy** is a Chrome extension designed to improve web accessibility for both color-blind users and web developers.
It helps users with color vision deficiencies browse websites more comfortably by enhancing visual clarity. At the same time, it assists developers in ensuring their websites meet **WCAG (Web Content Accessibility Guidelines)** for proper color contrast and readability.

## Features

### 👁️ User Mode (For Color-Blind Users)
- **Color Vision Assistance**  
  Enhances webpage colors to improve visibility for users with color vision deficiencies.
  Allows users to switch between different color vision modes for a personalized browsing experience.
  
### 🛠️ Developer Mode
- **WCAG 2.2 Compliance Check**  
  Ensures that web content meets accessibility standards by analyzing color contrast based on WCAG 2.2 guidelines.

  - **Auto Scan**  
    Automatically scans the entire webpage and identifies elements with insufficient color contrast.  
    Highlights problematic areas in real-time, allowing developers to quickly locate and fix accessibility issues without manual effort.

  - **Manual Check (Eyedropper Tool)**  
    Provides an interactive eyedropper tool that allows developers to select any foreground and background colors on the page.  
    Displays precise contrast ratios and indicates whether the selected color combination passes or fails WCAG standards, making it ideal for detailed inspection and fine-tuning.

- **Vision Simulation**  
  Simulates how a webpage appears to color-blind users in multiple modes (e.g., protanopia, deuteranopia, tritanopia).

- **Color Palette Extractor**  
  Extracts the color palette of any website automatically for analysis.

## Installation (Chrome Extension)
1.  **Download the repository**: Clone or download this GitHub repository to your local machine.
2.  **Open Chrome Extensions**: Navigate to `chrome://extensions` in your Chrome browser.
3.  **Enable Developer Mode**: Toggle on the **'Developer mode'** switch, usually located in the top right corner.
4.  **Load Unpacked**: Click on the **'Load unpacked'** button.
5.  **Select Directory**: Select the folder where you cloned/downloaded this repository.
6.  **Extension Ready**: The extension should now be installed and visible in your Chrome extensions list. You can pin it to your toolbar for easy access.
